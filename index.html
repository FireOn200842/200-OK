Existem várias maneiras de chegar a sítios ou fazer alguma tarefa no computador. Na vida real existem atalhos para determinadas situações, logo no computador também haverão. Aqui estão alguns exemplos:

Ctrl+c

Serve para copiar um ou mais items.

Ctrl+v

Serve para colar um ou mais items.

Ctrl+x

Serve para cortar um ou mais items.

Ctrl+g

Serve para guardar um documento.

Ctrl+t

Serve para selecionar todo o texto de uma página e/ou documento.

Ctrl+z

Serve para desfazer uma alteração.
